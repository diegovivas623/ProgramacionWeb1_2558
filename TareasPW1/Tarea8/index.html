Colores Multinavegador

Hay mas de 160 colores con nombre que son compatibles con todos los navegadores modernos. Estos nombres se pueden usar directamente en el archivo de estilos para aplicar colores a elementos sin especificar valores o modelos de color.
Por ejemplo:

  1. Negro
  2. Blanco
  3. Salmon
  4. Azul
  5. Verde

Colores de un borde

Para que un elemento de HTML pueda tener un color en su borde, son necesarias 2 propiedades, border style y border color, adicionalmente se puede modificar su tamaño utlizando border width
