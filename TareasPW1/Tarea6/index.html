Lenguaje de Marcado de Hipertexto

Bienvenidos hoy estaremos practicando algunos de los conceptos y elementos más importantes que se han visto hasta el momento en el curso de Programación Web 1 sobre HTML, además de la importancia de contendores y etiquetas semánticas en la estructuración de un sitio web.

Conceptos Principales de HTML

HTML

Lenguaje de Marcado de Hipertexto que se utiliza para crear la estructura de páginas web

Elemento HTML

Es la unidad básica de estructura y contenido de una página web, esta compuesta de:

  1. Etiqueta de Apertura
  2. Pares atributo-valor
  3. Contenido
  4. Etiqueta de Cierre

Etiqueta <head>

Es una etiqueta que se utiliza para comunicarse con el navegador a través de los metadatos, permite mejorar el SEO, además en ella se enlazan archivos externos de estilos o scripts

Etiqueta <body>

Todo el contenido HTML que el usuario va a visualizar

Contenedores Semanticos

<header> Representa el contenido introductorio de una página (a menudo contiene un logotipo, un menú de navegación u otra información de identificación del sitio).
<nav> Representa una sección de una pagína que contiene enlaces a otras paginas o secciones del sitio. Se usa para enlaces de navegación que ayudan a los usuarios a navegar por el sitio web.
<main> Representa el contenido principal de una página (no debe incluir ningun contenido que se repita en varias paginas como menus de navegación, pies de pagina o barras laterales).
<section> Representa una sección de una página que agrupa contenido relacionado dentro del sitio web (se puede utilizar para dividir una página en varias secciones o agrupar contenido dentro de una sección más grande).
<article> Representa una unidad autonoma de contenido o historia que puede distribuirse o reutilizarse de forma independiente por ejemplo publicaciones de blogs, articulos de noticias o reseñas de productos.
<footer> Representa el contenido de cierre de una página o sección / la parte inferior de una página o sección (a menudo contiene información de derechos de autor, detalles de contacto, ubicación u otra información complementaria).

Elementos Principales de HTML

Elemento <a>

Elemento que se utiliza para enlazar o navegar a otras páginas o secciones, hay 3 tipos de enlaces:

  • Externo
  • Interno
  • Anclaje (Navegar dentro de la misma página)

Algunos valores especiales en su atributo href son:

href="https://wa.me/12345687"
Este valor de atributo envia un whatsapp en un número en especifico
href="mailto:ianamalosextras@gmail.com"
Este valor se usa para enviar un correo a una dirección de correo especifica.

<form>

Es la manera en que los usuarios se pueden comunicar con una página web gracias a este elemento se pueden procesar datos de los usuarios y usarlos según las necesidades u objetivo de nuestro sito.
