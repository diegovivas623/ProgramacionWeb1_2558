Formas de agregar estilos a una página web(HTML)

Estilos en linea

Estos estilos se agregan directamente en el elemento HTML mediante el atributo style

Este parrafo tiene un fondo agregado con estilos en linea

Estilos internos

Estos estilos se agregan en el mismo archivo HTML dentro del elemento <style>, este elemento debe estar dentro de la etiqueta <head>

Estilos externos

Estos estilos son los recomendados ya que ofrecen modularidad, facil mantenimiento, orden, eficiencia entre otras ventajas para agregar una hoja de estilos externa, dentro de la etiqueta <head>, se va a utilizar la etiqueta <link>.
